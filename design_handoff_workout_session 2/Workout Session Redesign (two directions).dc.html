Workout Session — reinvented
A fresh visual language and two ways to track sets with far fewer taps. The old flow needed expand → tap field → type on a keypad → switch field → type → confirm. Both directions below pre-fill every set from last time, so the common case is one tap to log. Live rest timers get the spotlight they deserve.
