# Product Overview

Progressive Workout is a React Native fitness app built with Expo that helps users track and complete workout programs and challenges.

## Core Features

- **Workout Programs**: Multi-session structured workout programs with exercises, rest periods, and warmups
- **Challenges**: Goal-oriented fitness challenges with target reps and progress tracking
- **Exercise Library**: Built-in and user-created exercises with categories and icons
- **Progress Tracking**: Session completion, personal records, consistency streaks, and detailed analytics
- **Session Management**: Timer-based workout sessions with step-by-step guidance
- **Data Import/Export**: QR code sharing and program import functionality

## User Flow

1. Users select from built-in programs or create custom ones
2. Programs consist of multiple sessions with structured blocks (warmup, exercises, rest)
3. During sessions, users follow timed steps and log completed sets/reps
4. Progress is automatically tracked with personal records and consistency metrics
5. Users can view detailed progress analytics and share programs

## Target Platforms

- iOS and Android (React Native)
- Web (Expo Web)
- Cross-platform with responsive design
